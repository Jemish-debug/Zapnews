ZapNews - A News Feed


ZapNews is a dynamic news feed application that fetches and displays the latest news articles from various sources. It provides users with a clean, responsive interface to browse headlines, read summaries, and access full articles. The project focuses on real-time data fetching, user-friendly UI, and efficient rendering of news content.


Features

News Fetching Integration

  • Fetches news data using Python, by importing BeautifulSoup4 and Requests Library
  • Supports preference-wise filtering (e.g., Technology, Sports, Business).
  • Data collection from multiple sources for genre-based

Responsive & Interactive UI

  • Card-based layout for news articles.
  • Hover animations (e.g., scale effects, shadow enhancements).

Dynamic Content Loading

  • "Load More" button for pagination.
  • Skeleton loading while fetching data.
